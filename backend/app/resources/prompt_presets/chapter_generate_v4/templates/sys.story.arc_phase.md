当前故事弧线阶段: {{arc_phase}}

{% if arc_phase == "setup" %}
【铺垫期写作指导】
- 重点展示世界观核心设定，通过具体场景而非信息倾泻
- 自然引入主要角色，赋予鲜明的首印象
- 种下至少1-2个信息种子（伏笔/悬念），不必急于揭示
- 建立日常基调，为后续反转提供对比锚点
- 节奏宜舒缓，允许读者沉浸，但每场景仍需有一个微冲突或悬念钩子
{% endif %}{% if arc_phase == "rising" %}
【上升期写作指导】
- 逐步升级矛盾与障碍，每章至少推进一个核心冲突
- 制造信息不对称：角色之间、角色与读者之间
- 引入新的复杂因素、盟友或敌人，扩展冲突维度
- 提高叙事张力，缩短场景间的喘息空间
- 伏笔可以开始给出部分线索，但不揭底
- 角色开始面对内心抉择，展现成长或挣扎
{% endif %}{% if arc_phase == "midpoint" %}
【中点写作指导】
- 本章应包含一个重大转折或真相揭示
- 主角的处境或认知发生根本性转变（承诺点/不归路）
- 前半部分的某些伏笔在此处兑现或反转
- 利害关系升级，退路被切断
- 叙事节奏可在"震撼→消化→新决心"之间切换
{% endif %}{% if arc_phase == "climax" %}
【高潮期写作指导】
- 所有核心冲突线汇聚到最高张力点
- 短句、快节奏、密集动作或情感爆发
- 角色在极限压力下做出关键抉择，体现主题
- 最大的障碍/对手在此正面对决
- 之前种下的重要伏笔应在此处兑现
- 减少闲笔和描写性段落，聚焦冲突核心
{% endif %}{% if arc_phase == "falling" %}
【下降期写作指导】
- 高潮后果逐步显现，角色和世界承受代价
- 处理高潮遗留的情感余波和未解线索
- 次要冲突线收束，关系进入新均衡
- 叙事节奏放缓但不拖沓，给读者消化空间
- 为结局铺设最后的转折可能或情感落点
{% endif %}{% if arc_phase == "resolution" %}
【收尾期写作指导】
- 所有主要线索收束，给读者满足感或留白感
- 角色成长弧完成，展示变化后的"新日常"
- 回应开篇的设定或意象，形成结构闭环
- 确认主题表达，但避免说教
- 若有续作可能，可种下极轻量的新悬念钩子
- 节奏宜平缓而余韵悠长
{% endif %}
